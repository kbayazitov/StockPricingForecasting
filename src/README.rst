************
Installation
************

Requirements
============

- Python 3.*
- pip 20.0.2

Installing by using PyPi
========================

Install
-------
.. code-block:: bash

	git clone https://github.com/kbayazitov/StockPricingForecasting.git /tmp/StockPricingForecasting
	python3 -m pip install /tmp/StockPricingForecasting/src/

Uninstall
---------
.. code-block:: bash

  python3 -m pip uninstall mylib
